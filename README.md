Ответы на вопросы
Что делает fetch?
Fetch - это современный API для выполнения HTTP-запросов. Он возвращает Promise, который разрешается в Response объект, содержащий ответ сервера. Позволяет легко работать с REST API и получать данные с серверов.

Зачем нужны async/await?
Async/await - это синтаксический сахар над Promises, который делает асинхронный код более читаемым и похожим на синхронный. Позволяет избежать "callback hell" и упрощает обработку ошибок через try/catch.

Как работает localStorage?
localStorage - это встроенное в браузер ключ-значение хранилище, которое сохраняет данные даже после закрытия браузера. Данные хранятся в виде строк, поэтому для объектов используется JSON.stringify() и JSON.parse().
